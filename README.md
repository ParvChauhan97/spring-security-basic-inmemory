# 🔐 Spring Security - In-Memory Authentication

A beginner-friendly Spring Boot project demonstrating the fundamentals of **Spring Security** using **In-Memory Authentication**.

## 🚀 Features
- In-Memory Authentication
- Authentication & Authorization
- Role-Based Access Control
- Secured REST APIs

## 🛠️ Tech Stack
- Java
- Spring Boot
- Spring Security
- Maven
- In-Memory Authentication

## 🎯 Purpose
This project was built to understand Spring Security fundamentals before implementing database-based authentication.
